DnsWarden provides stable and privacy-focused dns resolvers.

Some features

  • NO Logs.
  • Supports multiple protocols like DNS-Over-TLS, DNS-Over-HTTPS and DNSCrypt.
  • Supports multiple types of dns filters. Like adblock, adultfilter and uncensored.
  • , where you can choose to block ads, spyware, malware and trackers.
  • Search to check whether a domain is present in filter list.
  • Hosted in multiple locations.
